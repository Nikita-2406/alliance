## В папке app размещено весь frontend приложения.

### Необходимые шаги для запуска приложения:
1. Скачать [Node.js](https://nodejs.org/en/download, "Хера смотришь, нажимай!")
2. Выполнить установку 
3. Перезагрузить ПК
4. Зайти в VSC и открыть консоль (ctrl + `)
5. Выполнить следущие команды 
   1. `npm install -g create-react-app` - установка фреймворка
   2. `npm install -g yarn` - установка пакетного менеджера
   3. Проверить установку пакетов с помощью командs `yarn -v`
   4. `cd app` - переход в рабочую папку
   5. `yarn` - установка всех нужных пакетов
   6. `yarn dev` - запускает сайт локально
   7. Сочитание клавиш `ctrl + c` остановит работу сайта
