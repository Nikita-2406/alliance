## 🎨 Эффекты дизайна

### Glassmorphism (на всех карточках)
```
background: rgba(255, 255, 255, 0.1);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.2);
box-shadow: 0 8px 32px 0 rgba(0, 0, 0, 0.3);
```

### Hover эффекты
- Подъем карточек: `transform: translateY(-4px)`
- Увеличение тени: `box-shadow: 0 12px 40px`
- Усиление свечения: `background: rgba(255, 255, 255, 0.15)`

### Анимации
- Fade In: появление контента вкладок
- Slide Down: раскрытие категорий
- Smooth transitions: все переходы 0.3s ease

## 📱 Навигация

### Маршруты
- `/` → Главная
- `/search` → Поиск
- `/categories` → Категории
- `/profile` → Профиль
- `/app/1` → PhotoMaster Pro
- `/app/2` → Fitness Tracker
- `/app/3` → Cloud Notes

### Способы навигации
1. Нижняя панель (4 основных раздела)
2. Клик на карточку приложения → страница приложения
3. Кнопка "Назад" → предыдущая страница
4. Ссылки "Все", "Показать все" → поиск/категории
